Model Verification Reports
==========================

This section contains mathematical verification reports for all RheoJAX model
families.  Each report compares the implemented equations against the published
literature to ensure correctness.

.. note::

   These reports are generated during the model verification campaign and serve
   as a permanent audit trail.  Each report documents the canonical equations,
   identifies any deviations, and provides references to the original papers.
