## Inference Workflow (Part B of 01-06 + NB09)

1. **Generate data** via `examples/utils/hvnm_demo_fit.make_synthetic_protocol_data(...)`
2. **Configure HVNM**: Set defaults, choose fittable parameters per protocol
3. **NLSQ fit**: Fast point estimation with warm-start
4. **NUTS sampling**: Bayesian posterior via NumPyro
5. **Diagnostics**: R-hat, ESS, trace plots, forest plots (ArviZ)
6. **PPC**: Posterior predictive checks overlaid on data
7. **Save**: JSON params, NPZ posteriors, CSV summary

## HVNM Architecture

4-subnetwork constitutive model for NP-filled vitrimers:

- **P** (Permanent): Covalent crosslinks, equilibrium modulus G_P
- **E** (Exchangeable): Vitrimer BER bonds, relaxation via TST kinetics
- **D** (Dissociative): Physical bonds, Maxwell-like relaxation
- **I** (Interphase): NP-matrix interface, independent BER kinetics

Key features: Guth-Gold X(phi) reinforcement, dual TST kinetics (matrix + interphase),
phi=0 recovers HVM exactly.
